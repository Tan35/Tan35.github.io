python流程控制语句

根据条件不同执行不同的代码是程序非常常见的也是非常重要的行为

布尔值

先来回顾一下关系运算符

符号 说明
> 大于
< 小于
>= 大于等于
<= 小于等于
== 相等
！= 不等

关系运算符最终得出的结果返回的就是布尔值(boolean)，而所谓的布尔值也就是False和True两个,比如说

  • print(12>5) # True
  • print(5>12) # False
  • True 和 False 是python里面的关键字（要么是真的要么是假的）

if 语句就是依靠布尔值来工作的
